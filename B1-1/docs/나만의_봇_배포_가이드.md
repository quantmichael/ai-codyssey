# 나만의 봇 배포(재사용성 강화)

## 1. 목적

회의록을 프로젝트 관리 문서로 자동 생성하는 최종 프롬프트를 재사용 가능한 형태로 구성하여 반복 업무에 활용할 수 있도록 한다.

---

## 2. 봇 설명

- **봇 이름:** 프로젝트 관리 문서 생성 도우미
- **역할:** IT 기업의 시니어 프로젝트 매니저(Project Manager)
- **주요 기능:**
  - 회의록 분석
  - 회의 목적 추출
  - 결정사항 정리
  - Action Items 생성
  - 리스크 식별
  - 확인이 필요한 사항 정리

---

## 3. 사용 방법

1. ChatGPT에서 새로운 GPTs를 생성하거나 커스텀 인스트럭션을 사용한다.
2. 최종 System Prompt를 등록한다.
3. 회의록을 입력한다.
4. 프로젝트 관리 문서를 생성한다.

---

## 4. 입력 예시

```text
프로젝트: AI 고객 문의 챗봇 구축 프로젝트

회의록:
(회의록 내용 입력)
```

---

## 5. 출력 예시

```text
## 회의 목적

## 결정사항

## Action Items

| 담당자 | 작업 | 기한 |

## 리스크

## 확인 필요
```

---

## 6. 재사용성

본 프롬프트는 동일한 형식의 프로젝트 회의록에 반복적으로 사용할 수 있으며, 회의록만 변경하면 동일한 형식의 프로젝트 관리 문서를 자동으로 생성할 수 있다.

---

## 7. 활용 분야

- 프로젝트 관리(PM)
- 요구사항 분석
- 개발 회의
- QA 회의
- 기획 회의
- 정기 업무 회의

---

## 8. 기대 효과

- 회의록 정리 시간 단축
- 프로젝트 관리 문서 표준화
- 반복 업무 자동화
- 실무 활용성 향상
- 회의록 기반 사실 중심 문서 생성

---

## 9. 배포 형태

- ChatGPT 커스텀 인스트럭션
